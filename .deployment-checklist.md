# 📋 Deployment Checklist

Використовуйте цей checklist для перевірки деплою.

## Pre-Deployment

### Локальна розробка
- [ ] Всі фічі працюють локально
- [ ] Немає console.log у продакшн коді
- [ ] Код пройшов lint: `npm run lint`
- [ ] Build проходить без помилок: `npm run build`
- [ ] Всі тести пройдені (якщо є)

### Безпека
- [ ] `.env` в `.gitignore`
- [ ] Жодних API ключів в коді
- [ ] Жодних хардкоднутих паролів
- [ ] AUTH_SECRET згенеровано: `openssl rand -base64 32`
- [ ] Перевірено вразливості: `npm audit`

### Git
- [ ] Всі зміни закоммічені
- [ ] Код запушено на GitHub
- [ ] README.md оновлено

---

## Railway Database

- [ ] PostgreSQL створено
- [ ] DATABASE_URL скопійовано
- [ ] Додано `?sslmode=require` в кінець URL
- [ ] Перевірено підключення локально
- [ ] Automatic backups увімкнено

---

## Vercel Deployment

### Проект
- [ ] Репозиторій підключено
- [ ] Framework preset: Next.js
- [ ] Root directory: `./`

### Environment Variables
- [ ] `DATABASE_URL` - з Railway
- [ ] `AUTH_SECRET` - згенеровано
- [ ] `AUTH_URL` - початково Vercel URL
- [ ] `NEXTAUTH_URL` - те саме що AUTH_URL
- [ ] `GEMINI_API_KEY` (опціонально)
- [ ] `ANTHROPIC_API_KEY` (опціонально)
- [ ] `OPENAI_API_KEY` (опціонально)

### Build & Deploy
- [ ] Build пройшов успішно
- [ ] Deployment працює
- [ ] Немає помилок в логах

---

## Database Migration

- [ ] Prisma міграція виконана: `npm run deploy:migrate`
- [ ] Таблиці створені
- [ ] Адмін акаунт створено: `npm run deploy:create-admin`
- [ ] Статус БД перевірено: `npm run deploy:db-status`

## Domain Setup

### Купівля домену
- [ ] Домен куплено на Nic.ua
- [ ] Домен активовано
- [ ] Доступ до DNS налаштувань

### Vercel
- [ ] Домен додано у Vercel
- [ ] DNS записи скопійовано

### Nic.ua DNS
- [ ] A запис додано: `@ → 76.76.21.21`
- [ ] CNAME запис додано: `www → cname.vercel-dns.com`
- [ ] DNS поширився (10-30 хв)
- [ ] SSL сертифікат виданий Vercel

### Update ENV
- [ ] `AUTH_URL` оновлено на новий домен
- [ ] `NEXTAUTH_URL` оновлено на новий домен
- [ ] Vercel redeploy виконано

## Cloudflare R2 (Optional)

  - [ ] `R2_ENDPOINT`

## Security

### Паролі
- [ ] Всі тестові паролі змінено
- [ ] Адмін пароль надійний (12+ символів)

### Access Control
- [ ] Права доступу працюють
- [ ] Клієнти не мають доступу до адмін панелі
- [ ] API захищено auth middleware
