```bash
cd /Users/admin/Igor-Shiba/metrum-group
PG18=/opt/homebrew/opt/postgresql@18/bin
```

## Stage 2: Baseline _prisma_migrations

```bash
PROD_URL=$(grep '^DATABASE_URL' .env | sed 's/DATABASE_URL=//;s/"//g')

# DRY RUN — побачити що буде записано
DATABASE_URL="$PROD_URL" node scripts/02-baseline-prod-migrations.mjs

# Якщо вивід виглядає коректно — реальний запис:
DATABASE_URL="$PROD_URL" node scripts/02-baseline-prod-migrations.mjs --confirm
```

Скрипт:
- Створює `_prisma_migrations` table
- Для кожної з 96 міграцій парсить її DDL → перевіряє чи tables/columns уже існують
- Якщо так → INSERT як applied
- Якщо ні → пропускає (буде застосовано через `migrate deploy`)

**Очікуваний результат:** ~93-94 міграцій marked applied, 2-3 залишаються для `migrate deploy` (RFI, Procurement, можливо SRM).

```bash
PROD_URL=$(grep '^DATABASE_URL' .env | sed 's/DATABASE_URL=//;s/"//g')
```

```bash
  --clean --if-exists --no-owner --no-acl \
```
