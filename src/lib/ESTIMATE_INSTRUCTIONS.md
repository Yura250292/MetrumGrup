# Інструкція з підрахунку кошторисів для будівельних проектів

## 1. Загальні принципи

### 1.1 Точність розрахунків
- Використовувати Decimal для всіх фінансових операцій
- Округлення: ціни до 2 знаків, кількість до 3 знаків
- Запас на підрізку матеріалів: 10-15%

### 1.2 Структура кошторису
- Мінімум 20 позицій для простих проектів
- 50-120+ позицій для комплексних проектів
- Групування за секціями (17 стандартних категорій)

## 2. Правила розрахунку площ

### 2.1 Площа стін
**Формула:** `S = периметр × висота`

Стандартна висота стелі: 2.7м (допускається 2.5-3.0м)

**Віднімання отворів:**
- Вікно стандартне: 1.5 м²
- Двері вхідні: 1.8 м² (0.9м × 2.0м)
- Двері міжкімнатні: 1.6 м² (0.8м × 2.0м)

**Приклад:**
```
Периметр кімнати: 20м (5м + 5м + 5м + 5м)
Висота: 2.7м
Вікна: 2 шт
Двері: 1 шт

S_стін = 20 × 2.7 - (2 × 1.5) - 1.8 = 54 - 3 - 1.8 = 49.2 м²
```

### 2.2 Площа підлоги та стелі
**Формула:** `S = довжина × ширина`

Запас на підрізку:
- Ламінат, паркет: +7-10%
- Плитка: +10-15%
- Лінолеум: +5-7%

### 2.3 Об'єм для бетону
**Формула:** `V = площа × товщина`

Приклад - стяжка:
```
Площа: 50 м²
Товщина стяжки: 5 см = 0.05 м
V = 50 × 0.05 = 2.5 м³
Запас на втрати: +5-10%
V_замовлення = 2.5 × 1.07 = 2.675 м³ ≈ 2.7 м³
```

## 5. Категорії робіт (17 стандартних секцій)

1. **Демонтажні роботи** - якщо потрібен демонтаж старих конструкцій
2. **Земляні роботи** - тільки для фундаментів
3. **Фундаментні роботи** - для нового будівництва
4. **Стіни та перегородки** - кладка, монтаж
5. **Перекриття** - монолітне, збірне
6. **Дах** - крокви, покрівля, водовідведення
7. **Вікна та двері** - демонтаж старих, монтаж нових
8. **Електрика** - проводка, розетки, освітлення
9. **Сантехніка** - труби, радіатори, змішувачі
10. **Стяжка підлоги** - вирівнювання, утеплення
11. **Штукатурні роботи** - стіни, стелі
12. **Шпаклювальні роботи** - вирівнювання під фарбу/шпалери
13. **Малярні роботи** - фарбування, лакування
14. **Плиткові роботи** - підлога, стіни (ванна, кухня)
15. **Підлогові покриття** - ламінат, паркет, лінолеум
16. **Оздоблення стель** - гіпсокартон, натяжні
17. **Інші роботи** - нестандартні позиції

## 6. Формули розрахунків

### 6.1 Розрахунок позиції кошторису
```
totalCost = (quantity × unitPrice) + laborCost
laborCost = laborHours × laborRate

Приклад:
  Штукатурка 50 м² × 180₴/м² = 9000₴
  Робота 50 м² × 1.5 год × 150₴/год = 11250₴
  Разом: 20250₴
```

### 6.2 Розрахунок секції
```
sectionTotal = Σ (totalCost всіх позицій секції)
```

### 6.3 Розрахунок підсумків
```
totalMaterials = Σ (quantity × unitPrice) всіх позицій
totalLabor = Σ laborCost всіх позицій
totalOverhead = (totalMaterials + totalLabor) × overheadPercent
totalBeforeDiscount = totalMaterials + totalLabor + totalOverhead
totalAmount = totalBeforeDiscount × (1 - discount/100)
```

Накладні витрати (overhead): 15-25% залежно від складності

### 6.4 Розрахунок маржі та податків
```
profitMargin = 20-30% (стандарт)
finalClientPrice = totalAmount × (1 + profitMargin/100)

Податки (для підприємця):
- ПДВ 20% (якщо платник ПДВ)
- ЄСВ 22% від зарплатного фонду
- ПДФО 18% + ВЗ 1.5% від зарплат
```

## 7. Типові помилки та як їх уникати

### 7.1 Математичні помилки

❌ **Помилка:** Неправильний розрахунок площі
```
Кімната 5×4м = 20 м² (підлога)
Стіни: 5+4+5+4 = 18м периметр × 2.7 = 48.6 м² (ЗАБУЛИ ВІДНЯТИ ВІКНА/ДВЕРІ)
```

✅ **Правильно:**
```
Стіни: 18м × 2.7м - 2×1.5м (вікна) - 1.8м (двері) = 48.6 - 3 - 1.8 = 43.8 м²
```

### 7.2 Помилки в матеріалах

❌ **Помилка:** Невраховані супутні матеріали
```
Плитка: 10 м² × 450₴ = 4500₴
(ЗАБУЛИ: клей, хрестики, затирку, грунтовку)
```

✅ **Правильно:**
```
1. Плитка: 10 м² × 1.15 запас × 450₴ = 5175₴
2. Клей Ceresit CM 11: 10 м² × 5 кг × 7₴/кг = 350₴
3. Затирка Ceresit CE 40: 2 кг × 180₴ = 360₴
4. Хрестики 2мм: 1 уп × 50₴ = 50₴
5. Грунтовка: 3л × 120₴ = 360₴
Разом матеріали: 6295₴
```

### 7.3 Помилки в працях

❌ **Помилка:** Нереалістичні норми часу
```
Штукатурка 50 м² за 1 день (8 год) = НЕРЕАЛЬНО
```

✅ **Правильно:**
```
Штукатурка машинна: 25-30 м² за 8 год (2 робітники)
Штукатурка ручна: 15-20 м² за 8 год (2 робітники)
50 м² машинна = 2 дні × 2 робітники = 4 людино-дні
```

### 7.4 Помилки в цінах

❌ **Помилка:** Ціни з 2020-2022 року
```
Газоблок D400: 60₴/шт (ціна 2020 року - ЗАСТАРІЛА)
```

✅ **Правильно:**
```
Газоблок D400: 85-95₴/шт (актуальна ціна 2025)
Джерело: https://epicentrk.ua/search/?q=газоблок+aeroc+d400
```

## 8. Чеклісти для перевірки

### Етап 1: Вхідні дані
- [ ] Площа коректно вказана (м²)
- [ ] Тип проекту обраний (foundation/shell/turnkey/...)
- [ ] Категорії робіт обрані відповідно до типу
- [ ] Файли завантажені та розпарсені

### Етап 2: Структура кошторису
- [ ] Кожна секція має логічну назву
- [ ] Позиції відсортовані в логічному порядку
- [ ] Мінімальна кількість позицій виконана (20+ для простих, 50+ для складних)
- [ ] Немає дублікатів позицій

### Етап 3: Математика
- [ ] Формула позиції: `totalCost = quantity × unitPrice + laborCost` ✓
- [ ] Всі одиниці вказані коректно (м², м³, м.пог, шт, кг)
- [ ] Кількість з запасом (10-15%)
- [ ] Підсумки секцій співпадають з сумою позицій
- [ ] totalMaterials + totalLabor + totalOverhead = totalBeforeDiscount

### Етап 4: Ціни та матеріали
- [ ] Всі ціни в діапазоні 2024-2025 років
- [ ] Ціни з посиланнями на пошукові URL (epicentrk.ua, prom.ua)
- [ ] Вказані конкретні марки (Ceresit, Knauf, Aeroc, ...)
- [ ] Вказані специфікації (розміри, вага, об'єм упаковки)
- [ ] Немає вигаданих посилань на товари

### Етап 5: Повнота
- [ ] Всі супутні матеріали враховані (клей, кріплення, грунтовка)
- [ ] Роботи відповідають матеріалам
- [ ] Інструмент та обладнання (якщо потрібно)
- [ ] Транспортні витрати (logisticsCost)
- [ ] Накладні витрати (15-25%)
