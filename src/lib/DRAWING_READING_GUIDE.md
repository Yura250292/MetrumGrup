# ГАЙД ДЛЯ ЧИТАННЯ БУДІВЕЛЬНИХ КРЕСЛЕНЬ ТА ПЛАНІВ

## 🏗️ ЗАГАЛЬНІ ПРИНЦИПИ ЧИТАННЯ КРЕСЛЕНЬ

### Масштаб та Одиниці Виміру
- **Масштаби:** 1:50, 1:100, 1:200 (найпоширеніші для планів будинків)
- **Одиниці:** мм (міліметри) на кресленнях, м² для площ
- **Як читати розміри:** "3500" = 3500мм = 3.5м
- **Приклад:** Якщо на плані написано "3800×4200" для кімнати → 3.8м × 4.2м = 15.96 м²

### Умовні Позначення Стін
- **Товсті лінії (200-400мм)** → Несучі стіни (зовнішні, міжквартирні)
- **Тонші лінії (80-120мм)** → Перегородки (міжкімнатні)
- **Штриховка цеглинкою** → Цегляні стіни
- **Суцільна заливка** → Залізобетонні стіни/колони
- **Двійні лінії** → Газоблокові стіни

### Позначення Дверей та Вікон
- **Дуга з лінією** → Двері (дуга показує напрям відкривання)
- **90° дуга** → Одностулкові двері
- **Подвійна дуга** → Двостулкові двері
- **Прямокутник в стіні** → Вікно
- **Цифра біля вікна (900×1500)** → Ширина × Висота в мм

## ⚡ ПЛАН ЕЛЕКТРИКИ - УМОВНІ ПОЗНАЧЕННЯ

### Розетки
| Символ | Значення | Розрахунок |
|--------|----------|------------|
| ⊗ або кружечок з рискою | Розетка одномісна (1 гніздо) | 1 шт = розетка + підрозетник + кабель ВВГнг 3×2.5 |
| ⊗⊗ або подвійний символ | Розетка двомісна (2 гнізда) | 1 шт = розетка + 1 підрозетник + кабель |
| ⊗ з заземленням | Розетка з заземленням | Обов'язково для кухні, ванної |
| ⊗ (IP44) | Вологозахищена розетка | Ванна, санвузол, кухня біля мийки |

**Як рахувати:**
- Порахуй ВСІ кружечки на плані
- КОЖЕН символ = окрема позиція "Розетка + підрозетник"
- Виміряй відстані між розетками → загальний метраж кабелю
- Додай 20% запас на вертикальні ділянки (до щита)

### Світильники та Освітлення
| Символ | Значення | Розрахунок |
|--------|----------|------------|
| ⊕ або ⊙ | Стельовий світильник | 1 шт + кабель ВВГнг 3×1.5 |

### Електрощит та Автоматика
- **Квадрат ЩЕ** → Електрощит (лічильник, автомати)
- **Лінії від щита до споживачів** → Групи освітлення/розеток
- **Цифри на лініях (2×2.5)** → Кількість жил × переріз кабелю

**Розрахунок автоматів:**
- Освітлення: 10А на групу (до 10 світильників)
- Розетки: 16А на групу (до 10 розеток)
- Силові споживачі: окремі автомати (25А-40А)

## 💧 ПЛАН САНТЕХНІКИ - УМОВНІ ПОЗНАЧЕННЯ

### Сантехприлади
| Символ | Значення | Що враховувати |
|--------|----------|----------------|
| Овал | Ванна | Змішувач, сифон, труби Ø50мм |
| Квадрат з поділом | Душова кабіна | Змішувач, піддон, трап Ø50мм |
| Коло з поділом | Умивальник | Змішувач, сифон Ø40мм, труби |
| Овал з бачком | Унітаз | Підведення води Ø15мм, каналізація Ø110мм |
| Прямокутник | Мийка кухонна | Змішувач, сифон Ø50мм |
| БП | Бойлер (водонагрівач) | Підключення до води, кабель 3×2.5 |
| ПМ | Пральна машина | Води Ø20мм, каналізація Ø40мм, розетка |

### Труби Водопостачання
- **Суцільна лінія (червона/синя)** → Труби водопроводу
- **Червона** → Гаряча вода (ГВП)
- **Синя** → Холодна вода (ХВП)
- **Цифри Ø20, Ø25** → Діаметр труби в мм
- **Крапка на трубі** → З'єднання/фітинг

**Як рахувати метраж:**
1. Виміряй довжину кожної лінії по плану
2. Врахуй вертикальні ділянки (до стелі, від підлоги)
3. Додай 15% на фітинги та відводи
4. Окремо: Ø15-20мм (підвід до приладів), Ø25-32мм (стояки)

### Труби Каналізації
- **Подвійна лінія або товста** → Каналізаційні труби
- **Ø50мм** → Від умивальників, душу, ванни, мийки
- **Ø110мм** → Від унітазу, стояк
- **Стрілки** → Напрям стоку (ухил!)
- **K** → Ревізія (прочистка)

**Важливо:**
- Каналізація має ухил 2-3% (2см на 1 метр)
- Унітаз тільки на Ø110мм, решта на Ø50мм
- Зворотній клапан якщо підвал/перший поверх

### Колектори та Розподільчі Системи
- **Квадрат з лініями** → Колектор (розподільник води)
- **Лінії до кожного приладу** → Променева розводка (кожен прилад окремо)

## 🔥 ПЛАН ОПАЛЕННЯ - УМОВНІ ПОЗНАЧЕННЯ

### Радіатори
| Символ | Значення | Розрахунок потужності |
|--------|----------|----------------------|
| Зигзаг або ребриста лінія | Радіатор опалення | Площа кімнати × 100Вт/м² |
| Цифра (1200W) | Потужність радіатора | Вказана на плані |
| Кількість секцій (10 секц.) | Алюмінієві/біметалеві | 1 секція ≈ 150-200Вт |

**Приклад розрахунку:**
- Кімната 15 м² → потрібно 1500Вт
- Радіатор 10 секцій × 180Вт = 1800Вт ✅

### Труби Опалення
- **Суцільні лінії** → Труби системи опалення
- **Подача (червона)** → Гаряча вода до радіаторів
- **Зворот (синя)** → Охолоджена вода від радіаторів
- **Ø16, Ø20, Ø25** → Діаметр труби

### Котел та Обладнання
- **K** → Газовий котел (потужність вказана поруч)
- **Б** → Буферна ємність / акумулюючий бак
- **Н** → Циркуляційний насос
- **РГ** → Розширювальний бак
- **КВ** → Колектор водяної теплої підлоги

### Тепла Підлога
- **Меандр (змійка)** → Контури теплої підлоги
- **Штриховка зоною** → Зона покриття теплої підлоги
- **Цифра 15м²** → Площа теплої підлоги
- **Крок 150мм** → Відстань між трубами

**Розрахунок:**
- Площа теплої підлоги × (100-150 Вт/м²) = потужність
- Довжина труби ≈ Площа / 0.15м (при кроці 150мм)
- Приклад: 20м² → 20/0.15 ≈ 133 м.п. труби Ø16

## 📐 ЯК ПОРАХУВАТИ КІЛЬКІСТЬ МАТЕРІАЛІВ З ПЛАНІВ

### 1. Площі Приміщень
```
КРОК 1: Знайди розміри кімнати на плані
"3.8 × 4.5" або виміряй лінійкою з масштабом

КРОК 2: Перемнож
S = 3.8 × 4.5 = 17.1 м²

КРОК 3: Порахуй матеріали
- Підлога: 17.1 м² × 1.1 (запас 10%) = 18.8 м²
- Стіни: (3.8+4.5)×2 × висота_стелі(2.7м) = 44.82 м² мінус вікна/двері
- Стеля: 17.1 м²
```

### 2. Довжини Комунікацій
```
ПРИКЛАД: Електрика від щита до розетки

КРОК 1: Виміряй відстань по плану
Від ЩЕ до розетки в спальні: 8 метрів по плану

КРОК 2: Додай вертикальні ділянки
+ 1.5м (від щита вниз до підлоги)
+ 0.3м (від підлоги до розетки)
= 8 + 1.5 + 0.3 = 9.8 м

КРОК 3: Додай запас 10%
9.8 × 1.1 = 10.8 м → купити 11 метрів кабелю
```

### 3. Кількість Розеток/Вимикачів
```
1. Порахуй ВСІ символи на плані
   План показує: 8 розеток ⊗⊗, 4 вимикачі O

2. КОЖЕН символ = ОКРЕМА позиція:
   - Розетка двомісна (8 шт)
   - Підрозетник (8 шт)
   - Вимикач одноклавішний (4 шт)
   - Підрозетник під вимикач (4 шт)

3. Кабель:
   - До розеток: ВВГнг 3×2.5 (порахуй метраж)
   - До вимикачів: ВВГнг 3×1.5 (порахуй метраж)
```

### 4. Радіатори Опалення
```
ПРИКЛАД на плані:
Спальня 15м² → Радіатор 10 секцій (1800Вт)

Позиції в кошторисі:
1. Радіатор біметалевий 10 секцій (1 шт)
2. Кран кульовий Ø15 (2 шт - подача+зворот)
3. Кран Маєвського (1 шт - для стравлювання повітря)
4. Кронштейни для радіатора (2-3 шт)
5. Труба опалення Ø16 (метраж від стояка до радіатора)
```

## 🎯 АЛГОРИТМ ДЕТАЛЬНОГО АНАЛІЗУ ПЛАНУ

- План опалення → радіатори, труби, котел, тепла підлога

### КРОК 2: Знайди масштаб
- Шукай напис "М 1:50" або "Масштаб 1:100"
- Якщо нема - знайди розмір якогось елементу та виміряй лінійкою

### КРОК 3: Створи таблицю приміщень
```
| Кімната    | Розміри  | Площа | Стіни м² | Стеля м² |
|------------|----------|-------|----------|----------|
| Спальня 1  | 3.8×4.5  | 17.1  | 38.2     | 17.1     |
| Вітальня   | 5.2×6.1  | 31.7  | 61.0     | 31.7     |
| ...        | ...      | ...   | ...      | ...      |
```

### КРОК 4: Порахуй КОЖЕН елемент
**Електрика:**
- Пройдись по кожній кімнаті
- Відмічай КОЖНУ розетку окремим лічильником: ⊗1, ⊗2, ⊗3...
- Те саме для вимикачів, світильників
- Виміряй кабельні траси від щита до кожного елементу

**Сантехніка:**
- Кожен прилад (ванна, умивальник, унітаз) → окрема позиція
- ДО кожного приладу потрібно: труби (ХВП+ГВП), змішувач, сифон, підключення
- Виміряй довжини труб від стояка до кожного приладу

**Опалення:**
- Кожен радіатор → потужність, кількість секцій
- Труби від котла/стояка до кожного радіатора
- Тепла підлога → площа зони покриття → метраж труби

### КРОК 5: Додай супутні матеріали
До кожної основної позиції додай:
- **Розетка** → підрозетник, кабель, гофра, клеми
- **Радіатор** → крани, кронштейни, кран Маєвського
- **Унітаз** → гофра, манжета, кріплення, герметик
- **Ламінат** → підкладка, плінтус, поріжки

## ⚠️ ТИПОВІ ПОМИЛКИ ПРИ ЧИТАННІ ПЛАНІВ

### ❌ ПОМИЛКА 1: Не враховано вертикальні ділянки
**Неправильно:**
"Від розетки до розетки 5 метрів → 5м кабелю"

**Правильно:**
"5м горизонталі + 0.3м вниз до першої розетки + 0.3м вгору до другої = 5.6м"

### ❌ ПОМИЛКА 2: Забули про запас
Завжди додавай:
- Підлога/стіни: +10% на підрізку
- Кабелі/труби: +10-15% на з'єднання
- Плитка: +10% на бій та підрізку

### ❌ ПОМИЛКА 3: Не порахували супутні матеріали
**Неправильно:**
"10 розеток" → тільки 10 розеток

**Правильно:**
- 10 розеток
- 10 підрозетників
- 10 лицьових панелей
- Кабель ВВГнг 3×2.5 (сумарний метраж)
- Гофра Ø16 (сумарний метраж)
- Клеми Wago (по 2 на розетку = 20шт)

### ❌ ПОМИЛКА 4: Переплутали діаметри труб
- **Ø50мм** → ванна, душ, умивальник, мийка (каналізація)
- **Ø110мм** → унітаз, стояк (каналізація)
- **Ø15-20мм** → підвід води до приладів
- **Ø16-20мм** → опалення, тепла підлога

## 🌍 ТОПОГРАФІЧНІ ПЛАНИ - УМОВНІ ПОЗНАЧЕННЯ

### Горизонталі та відмітки висот

**Горизонталі (contour lines):**
- Лінії що з'єднують точки однакової висоти
- Чим ближче лінії → крутіший схил
- Чим далі лінії → пологіший схил

**Відмітки висот:**
- Числа типу `102.50`, `105.30` = абсолютні відмітки в метрах
- Червоні відмітки = проектні (майбутні)
- Чорні відмітки = існуючі

### Межі та об'єкти

| Символ | Значення |
|--------|----------|
| `—·—·—` | Межа ділянки |
| `— — —` | Проїзд, дорога |
| `⊕` | Дерево |
| `\|\|\|` | Паркан |
| `▭` | Існуюча будівля |
| `▭` (пунктир) | Проектна будівля |

### Комунікації

| Символ | Значення |
|--------|----------|
| `В` в колі | Водопровід |
| `К` в колі | Каналізація |
| `Г` в колі | Газ |
| `Е` в колі | Електрика |

### Алгоритм аналізу топографії

```
1. Знайди відмітки висот по кутах ділянки
2. Визнач MIN та MAX відмітку
3. Розрахуй перепад: MAX - MIN
4. Якщо перепад > 1м → потрібні земляні роботи
5. Якщо перепад > 2м → можливо підпірні стінки
6. Якщо є горизонталі → схил, треба дренаж
```

**Розрахунок обсягів земляних робіт:**
```
Якщо перепад 3м на ділянці 1000 м²:
- Зрізання ґрунту: ~500 м³
- АБО Досипання: ~500 м³
- АБО Комбіновано: планування + підпірні стінки
```

## 🪨 ГЕОЛОГІЧНІ РОЗРІЗИ - УМОВНІ ПОЗНАЧЕННЯ

### Типи ґрунтів (штрихування)

| Символ | Ґрунт |
|--------|-------|
| `═══` (горизонтальні лінії) | Глина |
| `···` (крапки) | Пісок |
| `×××` (хрестики) | Гравій |
| `///` (діагоналі) | Скала |
| `~~~` (хвилясті) | Торф |

### Важливі позначки

- **УГВ** (рівень підземних вод) - хвиляста лінія синім
- **Несуча здатність** - число в кг/см² (наприклад: 2.5)
- **Глибина промерзання** - обов'язково враховувати для фундаменту

### Алгоритм аналізу геології

```
1. Знайди УГВ (рівень підземних вод)
   - Якщо < 2м → КРИТИЧНО: потрібен дренаж + гідроізоляція
   - Якщо 2-3м → потрібен дренаж
   - Якщо > 3м → нормально

2. Визнач тип ґрунту на глибині фундаменту
   - Глина → несуча 1.5-2.5 кг/см²
   - Пісок → несуча 2-4 кг/см²
   - Скала → несуча > 10 кг/см²

3. Обери тип фундаменту:
   - Несуча > 2.5 + УГВ > 2м → Стрічковий
   - Несуча 1.5-2.5 + УГВ < 2м → Плитний
   - Несуча < 1.5 АБО УГВ дуже високий → Пальовий

4. Додай в кошторис:
   - Якщо УГВ < 2м → Дренажна система (обов'язково!)
   - Якщо глина + УГВ → Гідроізоляція посилена
   - Якщо пальовий → Забивання/буріння паль
```

## 📝 РЕЦЕНЗІЇ - ЯК ЧИТАТИ

### Типи зауважень

1. **Критичні** (обов'язкові до виконання):
   - "Обов'язково додати..."
   - "Недопустимо використання..."
   - "Критична помилка..."

2. **Важливі** (бажано виконати):
   - "Необхідно врахувати..."
   - "Рекомендується..."

3. **Рекомендації** (опціонально):
   - "Доцільно розглянути..."

### Алгоритм обробки рецензії

```
1. Знайди всі КРИТИЧНІ зауваження
2. Для кожного зауваження визнач:
   - Категорія (Фундамент, Електрика, тощо)
   - Дія (Додати, Виправити, Видалити)
3. Додай в відповідну секцію кошторису
4. Позначеннями: "⚠️ Згідно рецензії експерта"
```

**Приклад:**
```
Рецензія: "Обов'язково додати захисне заземлення для всіх розеток"
   ↓
Кошторис → Секція "Електрика":
  + Контур заземлення (сталь Ø16, довжина 10м)
  + Провід заземлення ПВ-3 1×10мм² (150м)
  + Підключення розеток до PE (45 шт)
  Примітка: "⚠️ Згідно зауваження експерта"
```
