## 🔧 Використання в Коді

### Базове використання

```typescript
import { calculateLaborCost } from '@/lib/labor-cost-calculator';

// Розрахунок вартості роботи
const result = calculateLaborCost(
  'Штукатурення стін',
  100, // м²
  'м²'
);

console.log(result);
// {
//   workName: 'Штукатурення стін',
//   quantity: 100,
//   unit: 'м²',
//   laborCost: 29500, // 100 × 295 грн/м²
//   source: 'kapitel_2025',
//   confidence: 0.95,
//   rate: { name: 'Штукатурення стін', price: 295, ... }
// }
```

### Пріоритет джерел

Функція `calculateLaborCost` шукає ціни в такому порядку:

1. **KAPITEL Прайс** (впевненість 0.85-0.95)
   - Найточніші та актуальні ціни
   - Перевірені реальною компанією

2. **База робіт** (впевненість 0.80)
   - Загальні ринкові ціни
   - Оновлюються регулярно

3. **Оцінка** (впевненість 0.40-0.60)
   - Коли немає в базах
   - Використовує середні по категорії

### Batch розрахунок

```typescript
import { calculateLaborCostsBatch } from '@/lib/labor-cost-calculator';

const works = [
  { name: 'Стяжка підлоги', quantity: 50, unit: 'м²' },
  { name: 'Штукатурення стін', quantity: 120, unit: 'м²' },
  { name: 'Монтаж розеток', quantity: 30, unit: 'од.' }
];

const results = calculateLaborCostsBatch(works);
const totalCost = results.reduce((sum, r) => sum + r.laborCost, 0);

console.log(`Загальна вартість робіт: ${totalCost} грн`);
// Загальна вартість робіт: 52,660 грн
// (50×290 + 120×295 + 30×92 = 14,500 + 35,400 + 2,760)
```

## 🎯 Інтеграція з AI Агентами

Агенти автоматично використовують розцінки KAPITEL:

```typescript
// В finishing-agent.ts, plumbing-agent.ts, electrical-agent.ts і т.д.

protected async estimateLaborCost(
  workName: string,
  quantity: number,
  unit: string
): Promise<number> {
  const result = calculateLaborCost(workName, quantity, unit);

  if (result.confidence < 0.7) {
    this.warnings.push(
      `⚠️ Низька впевненість в розцінці для "${workName}": ${result.source}`
    );
  }

  return result.laborCost;
}
```

### Переваги:

✅ **Актуальні ціни** - прайс від 08.09.2025  
✅ **Реальна компанія** - KAPITEL (Львів/Івано-Франківськ)  
✅ **77 позицій робіт** - охоплює основні категорії  
✅ **Автоматичний fallback** - якщо немає в Kapitel → шукає в базі → робить оцінку  
✅ **Confidence рівень** - AI знає наскільки точна ціна

## 📝 Примітки

1. **Одиниці виміру:**
   - `м²` - квадратні метри
   - `м/п` або `м.п.` - погонні метри
   - `т.` - точка (1 точка = 1 елемент/підключення)
   - `од.` - одиниця (штука)
   - `т/пов.` - тонна/поверх

2. **"Від X грн"** - означає мінімальну ціну, фактична може бути вищою в залежності від складності

3. **Географія:** Ціни актуальні для Західної України (Львів, Івано-Франківськ). Для Києва можливо +10-15%.
